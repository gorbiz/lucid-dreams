# Transerver
Web server serving Gists & GitHub repos,  
also supports [Jade](http://jade-lang.com/) & [Stylus](http://stylus-lang.com/).

Supported forms:
```
http://host/GIST_ID/[filename]
http://host/USER/REPO/[filename]
```
